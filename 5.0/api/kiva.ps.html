draw_image(img_gc, rect=(x,y,w,h))

Draws another gc into this one. If ‘rect’ is not provided, then the image gc is drawn into this one, rooted at (0,0) and at full pixel size. If ‘rect’ is provided, then the image is resized into the (w,h) given and drawn into this GC at point (x,y).

img_gc is either a Numeric array (WxHx3 or WxHx4) or a PIL Image.

Requires the Python Imaging Library (PIL).

Draws text on the device at the current text position.

This relies on the FreeType engine to render the text to an array and then calls the device dependent device_show_text() to display the rendered image to the screen.
