kiva.image module¶

This backend supports Kiva drawing into memory buffers.

Though this can be used to perform drawing in non-GUI applications, the Image backend is also used by many of the GUI backends to draw into the memory space of the graphics contexts.

kiva.image.font_metrics_provider()[source]¶

Create an object to be used for querying font metrics.
